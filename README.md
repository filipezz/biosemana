# Express Boilerplate

Boilerplate pré configurado para aplicações express.

## Features
 - Prettier
 - ESlint
 - Sequelize
 - Nodemon
 - VSCode debug

## Requerimentos

 - [Node](https://nodejs.org/en/download/current/)  
 - [Yarn](https://yarnpkg.com/en/docs/install)
 - [Docker](https://www.docker.com/)
 
 #### Instalar dependências:

```bash
yarn
```

#### Rodando localmente

```bash
yarn dev
```
